Jupyter Notebook: Caso Práctico

Una vez instalado Jupyter Notebook, es importante conocer los conceptos básicos para trabajar con él. Este tutorial te enseña cómo crear, editar y ejecutar celdas en Visual Studio Code.

💻 En Visual Studio Code


Tipos de celdas: Code y Markdown

En Jupyter existen dos tipos principales de celdas:

  • Code: Celdas donde escribes código Python ejecutable. El código se ejecuta cuando presionas el botón "Run" o Ctrl+Enter, y los resultados se muestran debajo de la celda.
  • Markdown: Celdas para escribir texto formateado, documentación y explicaciones. Puedes usar markdown para crear títulos, listas, tablas, hipervínculos y más. Se renderizan cuando ejecutas la celda.

Para cambiar el tipo de una celda, haz clic en el desplegable de tipo de celda en la barra superior de VSCode.

Interfaz de celdas en VSCode

  • Cada celda tiene un botón "Run" (triángulo) a la izquierda
  • Haz clic para ejecutar esa celda
  • O presiona Ctrl+Enter para ejecutar
  • Los resultados aparecen debajo de la celda automáticamente

Panel de variables en VSCode

  • VSCode muestra automáticamente un panel con las variables definidas
  • Puedes ver el valor de todas las variables en memoria
  • Útil para debugging y entender el estado del kernel
  • El panel se actualiza cada vez que ejecutas una celda

⚙️ Características útiles en VSCode

Ejecutar varias celdas

  • Usa "Run All" en la barra superior para ejecutar todas las celdas
  • O "Run All Above" para ejecutar todas las anteriores
  • O "Run All Below" para ejecutar todas las posteriores

Enviar selección a terminal

  • Selecciona código en una celda
  • Presiona Shift+Enter para ejecutar solo la selección
  • Útil para ejecutar partes específicas del código

Reinstiar kernel

  • Si el kernel falla o quieres limpiar el estado, haz clic en el botón "Restart" en la barra
  • O presiona Ctrl+Shift+F5
  • Esto borrará todas las variables en memoria

📊 Ejemplo práctico: Análisis básico de datos


Ejemplo: Crear un gráfico simple

# En caso de no tener instalada alguna librería se hace este paso, si no, se omite

%pip install pandas numpy matplotlib

#------------------------------------------------------------------

import pandas as pd

import matplotlib.pyplot as plt

# Datos de ventas por mes

meses = ['Enero', 'Febrero', 'Marzo', 'Abril', 'Mayo']

ventas = [100, 150, 120, 180, 200]

# Crear DataFrame

df = pd.DataFrame({'Mes': meses, 'Ventas': ventas})

print(df)

# Crear gráfico

plt.plot(df['Mes'], df['Ventas'], marker='o')

Tips importantes:

  • Siempre documenta tu trabajo con celdas Markdown
  • Organiza tu notebook en secciones lógicas
  • Ejecuta las celdas en orden para evitar confusiones
  • Reinicia el kernel regularmente si trabajas con muchas variables
  • Guarda frecuentemente con Ctrl+S
