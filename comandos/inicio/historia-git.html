Historia de Git

Git es un sistema de control de versiones distribuido creado por Linus Torvalds en 2005 para gestionar el desarrollo del kernel de Linux. Su diseño descentralizado, rápido y seguro solucionó las limitaciones de sistemas previos como CVS o Subversion, permitiendo a los desarrolladores trabajar de forma autónoma y colaborar eficientemente sin depender de un único servidor central.

En 2008 nació GitHub como una plataforma web que alojaba repositorios Git e incorporaba funciones colaborativas como issues, pull requests y wikis. Gracias a su interfaz sencilla y sus herramientas sociales, GitHub facilitó la colaboración abierta y masiva en proyectos de software, convirtiéndose en el lugar de encuentro para millones de desarrolladores.

Tras su adquisición por Microsoft en 2018, GitHub ha seguido creciendo, incorporando herramientas como Actions, Codespaces y Copilot, consolidándose como la plataforma de referencia para el desarrollo de software colaborativo, tanto en proyectos de código abierto como privados.
