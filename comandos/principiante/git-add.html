git add

Prepara cambios para el próximo commit. Puedes añadir un solo archivo o todos los cambios pendientes

⌨️ En terminal


    Dentro de la carpeta la cual estamos trabajando ejecutamos lo siguiente:

    git add .

    Con el "." lo que estamos diciendo a git es que añada la carpeta en la que estamos, también podemos añadir solo 1 archivo o carpeta dentro de la carpeta en la que estamos, poniendo lo siguiente en vez del punto:

    git add (carpeta o archivo a seleccionar)
